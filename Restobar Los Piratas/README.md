# Los Piratas Restobar

Sitio web informativo para **Los Piratas Restobar**, desarrollado como proyecto del curso **Taller de Programación Web**.

## Descripción

El proyecto presenta una web temática para un restobar con identidad visual pirata. Incluye páginas de inicio, menú, promociones, nosotros y contacto.

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Google Fonts

## Páginas del sitio

- `index.html`: portada principal del restobar.
- `menu.html`: carta con ceviches, bebidas y parrillas.
- `promociones.html`: promociones destacadas.
- `nosotros.html`: historia, misión y visión.
- `contacto.html`: datos de contacto y formulario.

## Paleta de colores

- Negro pirata: `#0B0B0B`
- Marrón oscuro: `#2B140C`
- Marrón madera: `#5C2A14`
- Cobre: `#A64B22`
- Dorado: `#D9A441`
- Pergamino: `#F3D2B8`
- Crema: `#FFF3E0`

## Tipografías

- **Pirata One** para títulos principales.
- **Poppins** para textos generales y navegación.

## Características implementadas

- Navegación entre páginas.
- Diseño responsive con media queries.
- Maquetación con Flexbox y CSS Grid.
- Tarjetas de productos y promociones.
- Uso de pseudo-clases como `:hover` y `:focus`.
- Uso de pseudo-elementos como `::before` y `::after`.
- Paleta visual coherente con la temática pirata.
